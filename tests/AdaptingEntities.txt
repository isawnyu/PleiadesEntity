AdaptingEntities.txt
====================

Test adaptation of GeographicEntity objects to IGeoItemSimple

A lot of this is cribbed from optilude's tutorial on plone.org.

    >>> from Products.PleiadesGeocoder.interfaces import IGeoItemSimple

Setup the test fixture

    >>> folder = self.folder

Add one blank entity to the folder, and adapt

    >>> folder.invokeFactory('GeographicEntity', id='aaa')
    'aaa'
    >>> item = IGeoItemSimple(folder.aaa)
    >>> item.isGeoreferenced()
    False
    >>> item.getInfo()
    {'srs': 'EPSG:4326', 'geometryType': 'point', 'description': '', 'spatialCoordinates': '', 'url': 'http://nohost/plone/Members/test_user_1_/aaa', 'title': 'aaa', 'id': 'aaa'}
